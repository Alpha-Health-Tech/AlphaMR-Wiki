Create and manage clinical documentation for patient consultations.

## Creating clinical notes

1. Open the patient's record
2. Navigate to **Clinical Notes**
3. Click **New Note**
4. Select the note template (if applicable)
5. Enter your clinical observations
6. Save the note

## Note templates

Use templates to standardize your documentation:

- Initial consultation
- Follow-up visit
- Procedure notes
- Progress notes

## Attaching files

You can attach supporting documents:

1. Open an existing note or create a new one
2. Click **Attach File**
3. Upload images, PDFs, or other documents
4. Save
